Como gerar o executável distribuível para meu sofware?

O procedimento é muito simples

  1. Compile o executável como "release"
  2. Abra o shell do Qt
  3. Dê o comando: windeployqt "pasta para o executável" (windows)
    macdeployqt "pasta para o executável" (Mac Os X)
    Para Linux entre no site Qt Documentation para ver um tutorial rápido.
    Exemplo: windeployqt "C:\Users\Jose\Desktop\release"
  4. Apage as pastas e arquivos inúteis: "translations", "qt5openglw.dll" ...
  5. Use um compilador de auto-executáveis como por exemplo Inno Setup Compiler
  6. Fim :)
